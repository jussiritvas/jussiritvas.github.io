Parallel Programming Runtimes

My current project under construction. The idea here is to provide a consol-based visualisation on the runtimes of different matrix operations with respect to different levels of parallelism.

The project demonstrates different levels of parallelism on both CPU and GPU. The CPU parallelisation is done with openMP and GPU version is done by using CUDA. The program is planned to visualise the 7 levels of CPU parallism and 3 levels of GPU parallelism.

The main reason for making the project was for me to understand parallelism better. The project is currently private but will be published here after it is ready.
